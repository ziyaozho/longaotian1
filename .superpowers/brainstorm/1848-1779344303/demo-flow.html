演示流程脚本 — 3分钟版

时间线 + 关键节点 + 每个阶段展示的亮点

整体节奏

时间分配：AI 生成 (40%)  |  叙事 (30%)  |  玩法 (30%)
开局 45s
回合1 40s
回合2 40s
高潮 30s
收尾 25s

总计约 3 分钟，留 1-2 分钟给旁白介绍和评委 Q&A

阶段 1：开局 (0:00-0:45) — 叙事 + 世界观建立

展示内容

  • 角色创建页 → 点击"演示开局"按钮
  • 自动填入最优属性 (天赋9/智力7/体质6...)
  • 世界选择 → 修仙世界（视觉效果最炫）
  • 系统绑定 → 吞噬进化系统（传说品质，最吸睛）
  • 系统对话触发 → 展示 AI 人格阶段变化

旁白要点

  • "这是一个 AI 驱动的系统流人生模拟器"
  • "每个世界都有独立的叙事弧线"
  • "注意系统的语气——它会随着你的旅程进化"

阶段 2：第一回合 (0:45-1:25) — AI 实时生成

展示内容

  • AI 生成中... 打字机动画展示 DeepSeek 实时输出
  • 场景描述 + 4个动态选项出现
  • 旁白解释："这些选项不是预设的，是AI根据你的属性实时生成的"
  • 选择一项 → 属性变化浮动数字动画
  • 触发随机事件 → 战斗遭遇

技术亮点标注

  • 🔧 DeepSeek API 实时调用
  • 🔧 上下文窗口管理（携带角色状态+世界观）
  • 🔧 本地模板回退机制（展示容错）

阶段 3：第二回合 (1:25-2:05) — 系统深度

展示内容

  • 系统面板弹出 → 展示吞噬进化升级树
  • 任务系统 → 主线任务进度更新
  • 物品栏 → 战斗获得的战利品
  • 成就解锁动画 → "初入修仙"成就
  • 场景过渡 → 从凡人界突破到练气期

玩法系统清单

  • 🎮 10个系统 (含传说隐藏)
  • 🎮 30+ 成就 + 元进度
  • 🎮 战斗/任务/物品/NPC 遭遇

阶段 4：高潮 (2:05-2:35) — 叙事爆点

展示内容

  • 系统对话进入情感阶段："宿主...我们已经走过了两个世界"
  • 世界树叙事触发 → "万界行者"身份揭示
  • 关键 NPC 遭遇 → 虚空吞噬者爪牙
  • 选择：牺牲自己保护世界 vs 逃离 → 展示分支后果

情感冲击点

  • 💎 系统从冷冰冰到有感情的进化
  • 💎 "万界行者"的世界观揭示
  • 💎 选择的重量感

阶段 5：收尾 (2:35-3:00) — 留白 + 引导

展示内容

  • 场景渐黑 → 世界树画面过渡
  • "你的冒险才刚刚开始..."
  • 游戏结束页 → 显示解锁内容预览
  • 8个世界、10个系统、5个隐藏内容
  • 二维码 → 体验完整游戏

评委记忆点

  • ✨ AI 实时生成不是噱头，是核心玩法
  • ✨ 内容深度超出预期（手写叙事层）
  • ✨ 技术完成度高（生产构建就绪）
