腾讯云 COS 部署步骤（推荐）

步骤 1：创建 COS 存储桶 → 开启静态网站托管
步骤 2：pnpm build → 上传 dist/ 到存储桶
步骤 3：绑定自定义域名 / 使用默认域名
步骤 4：配置 CDN 加速（可选但推荐，演示流畅度加分）
✅ 完成：获得国内可直接访问的 URL + 二维码

一键部署脚本

scripts/deploy.sh — 构建并推送到国内 CDN
#!/bin/bash
# 部署到腾讯云 COS
echo "🔨 构建生产版本..."
pnpm build

echo "📤 上传到 COS..."
coscmd upload -r dist/ /

echo "🧹 刷新 CDN 缓存..."
coscmd purge /

echo "✅ 部署完成！"
echo "📍 访问: https://your-bucket.cos.ap-guangzhou.myqcloud.com"
echo "📱 二维码已生成: qrcode.png"
